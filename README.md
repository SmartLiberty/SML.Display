# SML.ExampleGrpc
Template Project for a SML.Worker Microservice

Steps to create worker project with this template:
- Click the button "Use This template" in this page to create the Github Repository
- Chose your repository name and create the repository
- Github will create a copy of SML.ExampleGrpc
- On your machine, clone the new repository, the one you just created (DON'T do Git-flow yet).
- Open Powershell in the folder for your repository and run the command: ".\RenameTemplateFiles.ps1"
- The PowerShell will ask you what is the Project name, insert your project name (For example: For my repository "SML.BlackBox")
- The script will rename all references to the template name.
- Commit your changes and push the code to master.
- ONLY NOW you should do Git-flow.

Now, you should have a usable template for a Worker Microservice
